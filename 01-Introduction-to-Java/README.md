# 1. Introduction to Java

## What is Java?

Java is a high-level, object-oriented, platform-independent programming language developed by Sun Microsystems in 1995. It is now owned by Oracle Corporation.

### Why Learn Java?

- Easy to learn
- Platform Independent
- Object-Oriented
- Secure
- Widely used in industry
- Used for Web, Desktop, Android, and Enterprise Applications

# 3. JDK, JRE, JVM

## JDK (Java Development Kit)

JDK is used to develop Java applications.

It contains:
- JRE
- Compiler (javac)
- Debugger
- Development Tools

---

## JRE (Java Runtime Environment)

JRE provides the environment to run Java programs.

It contains:
- JVM
- Libraries
- Supporting Files

---

## JVM (Java Virtual Machine)

JVM executes Java Bytecode and makes Java platform independent.

Responsibilities:
- Loads class files
- Executes bytecode
- Manages memory
- Garbage Collection
